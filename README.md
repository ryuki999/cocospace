※動作確認に際してユーザIDやパスワードなどは表示されている

### 実装・機能
* **メール認証機能**
  * 新規登録時にメールアドレスを入力して仮登録(24時間以内に登録しないと自動破棄)。
  * メールアドレス宛にURLが送付されるのでクリック。
  * ユーザIDとパスワードを入力・確認して登録環境。
* **キャッシュ機能**
  * 一度ページを閉じてもブラウザを閉じなければ自動的にログインされる
* 画像/動画/コメントの投稿・削除・編集
  * **自分の好きな内容を投稿可能！**

### 開発環境
* PHP
* MySQL
* Smarty：テンプレートエンジン
